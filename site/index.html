Cross Platform

Built using Kotlin Multiplatform targeting Android, iOS, JVM, and Web (Wasm). A unified codebase handling the platform's core health functionality seamlessly.

Open Documentation

All API designs and markdown specifications are deeply integrated. They share the exact same responsive styling and theme as this landing page.

No-JS Theming

Lightning fast page load. All theming and responsiveness are implemented with pure HTML5 and modern CSS techniques, utilizing standard CSS variables.

Modern Architecture

Utilizing SQLDelight for local database persistence across platforms, and Compose Multiplatform for consistent declarative UI.
